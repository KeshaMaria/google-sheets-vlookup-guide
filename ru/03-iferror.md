## Этап 3. Обработка ошибок для товаров, которых нет в Прайс-листе: заполнить колонку Цена значением «нет в наличии».

**Если товара нет**, в ячейке появится значение «#N/A». Чтобы заменить «#N/A» на значение «нет в наличии», добавьте в формулу функцию ЕСЛИОШИБКА.

```excel
=ЕСЛИОШИБКА( Формула ВПР + условие «ЕСЛИ» ; "нет в наличии" )
```

*ЕСЛИОШИБКА*

```excel
=ЕСЛИОШИБКА( ВПР( A2 ; 'Прайс-лист'!A:C ; ЕСЛИ( B2="Розница" ; 2 ; 3 ) ; 0 ) ; "нет в наличии" )
```
*Формула ЕСЛИОШИБКА + ВПР + Условие «ЕСЛИ»*
